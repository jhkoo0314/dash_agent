## 1. Executive Summary
이번 리서치의 결론은 **“엑셀로 2주 내 증명 가능한 반복 운영업무”**가 에이전트 도입의 1순위라는 점입니다.  
Top-5는 `KPI/실행률 리포팅`, `콜플랜 실행 모니터링`, `CRM 데이터 품질 진단`, `인센티브 정산 사전검증`, `MLR 콘텐츠 운영 병목 관리`입니다.

선정 이유는 공통적으로:
1. 월/주 반복 빈도 높음
2. 본부 병목(취합/검증/질의 대응) 존재
3. 고정 산출물(리포트/가이드/FAQ) 존재

특히 에이전트는 **규칙기반 예외판정 + 코멘트 생성 + 질의응답**까지 처리하여 운영 효율이 극대화됩니다.

## 3. Top-3 Excel-first Agent Blueprint

### Blueprint A: KPI/실행률 통합 리포팅 에이전트
* **목적**: 주간 KPI 자동 집계 및 이탈 원인 코멘트 표준화.
* **입력**: `activity_log`, `target_plan`, `sales_kpi`, `org_master`.
* **출력**: 현장용 1p(우선 계정), 본부 요약(팀별 리스크), 진단 리포트.
* **운영**: Power Query + 피벗 + 에이전트 코멘트 루틴.

### Blueprint B: CRM 데이터 품질 진단 에이전트
* **목적**: 리포트 왜곡의 주범인 조인 실패/중복 주간 차단.
* **입력**: `hcp_master`, `call_log`, `territory_map`.
* **출력**: 수정 필요 계정 리스트, DQ Score 대시보드.

### Blueprint C: 인센티브 사전검증 에이전트
* **목적**: 지급 전 오류 축소 및 질의 대응 시간 단축.
* **입력**: `ic_rulebook`, `payout_rate`, `exception_log`.
* **출력**: 개인 정산 요약, 오류 히트맵, 표준 답변 템플릿.
