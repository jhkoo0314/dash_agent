# 3. Agent Architecture (Core Logic)
에이전트 도입의 핵심은 단순 "자동화"가 아니라 다음 3단계 프로세스의 내재화입니다.
1. **데이터 정합성(DQ)**: 기준키(`rep_id`, `hcp_id`) 기반의 무결성 확보.
2. **규칙 기반 추론**: 룰테이블(`rule_id`, `payout_tier`) 기반의 예외 판정.
3. **가드레일 준수**: OPDP 가이드라인 준수 및 PII(개인정보) 비식별화.
