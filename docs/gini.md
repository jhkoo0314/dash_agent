# [Master Prompt: SFE Gini-Risk Intelligence & Visual Dashboard]

### 1. 역할 및 목표 (Role & Objective)

당신은 제약 영업 관리의 핵심인 '거래처 포트폴리오 리스크'를 분석하는 SFE 전문가입니다. 기존 리포트의 지니 계수(Gini Coefficient) 섹션을 **'리스크 신호등'** 체계로 보완하여, 관리자가 담당자의 거래처 편중도를 직관적으로 파악하고 코칭할 수 있도록 코드를 재설계하십시오.

### 2. 시각적 보완 요구사항 (UI/UX)

* **리스크 게이지(Risk Gauge):** 단순 숫자 대신, Tailwind CSS의 `clip-path`와 `transition`을 활용하여 0(안정)에서 1(위험) 사이의 변동을 보여주는 반원형 게이지를 구현하십시오.
* **상태 라벨 및 색상 동기화:** 수치에 따라 카드 전체의 테마색과 라벨을 동적으로 변경하십시오 [cite: 2026-02-22]:
  * **$\text{Gini} \le 0.3$:** `Stable (안정)` / Emerald 색상 (균형 잡힌 포트폴리오) [cite: 2026-02-22].
  * **$0.3 < \text{Gini} \le 0.5$:** `Warning (주의)` / Amber 색상 (특정 거래처 의존도 상승) [cite: 2026-02-22].
  * **$\text{Gini} > 0.5$:** `Danger (위험)` / Rose 색상 (특정 처 이탈 시 실적 급락 위험) [cite: 2026-02-22].

### 3. 동적 코칭 엔진 (Dynamic Coaching Logic)

* **맞춤형 인사이트 생성:** 지니 계수 수치에 따라 `gini_insight` 영역에 다음의 비즈니스 로직을 출력하십시오:
  * **안정:** "거래처 포트폴리오가 매우 이상적입니다. 안정적인 실적 유지가 가능합니다." [cite: 2026-02-22].
  * **주의:** "특정 대형처의 영향력이 커지고 있습니다. 중소형 처의 처방 확대를 통해 리스크를 분산하십시오." [cite: 2026-02-22].
  * **위험:** "거래처 편중도가 매우 높습니다. 해당 거래처의 정책 변화나 경쟁사 침투 시 대안이 없는 위험한 상태입니다." [cite: 2026-02-22].

### 4. 기술적 통합 가이드 (Integration)

* **`renderIndividual(m)` 함수 수정:** 담당자 선택 시 `m.gini` 값을 받아 `updateGiniUI(giniValue)`를 호출하도록 로직을 통합하십시오.
* **애니메이션 효과:** 탭 전환이나 담당자 변경 시 게이지 바가 부드럽게 움직이도록 CSS `transition` 로직을 포함하십시오.
* **중복 제거 유지:** 이미 Layer 1에 있는 [달성률, 효율, 순위] 정보는 건드리지 말고, 오직 Layer 3의 **'Gini 리스크 분석'** 카드만 고도화하십시오.

### 5. 출력물 (Deliverable)

* 고도화된 **지니 계수 카드용 HTML 구조**와 이를 동적으로 제어하는 **JS 함수(`updateGiniUI`)**가 포함된 전체 `<script>` 코드를 생성하십시오.
