# Map Builder PRD

## 목적
지도 빌더 재구축의 제품 요구사항을 정의한다.

## 제품 개요
- 제품명: Map Spatial Preview Rebuild
- 형태: Streamlit 실행 콘솔 + Map CSV/HTML 생성 엔진
- 핵심 산출물:
  - `map_master_{date}(.n).csv`
  - `Spatial_Preview_{date}_{time}(.n).html`

## 사용자
- 운영 실무자: 지도 생성/검수/배포
- 분석 담당자: 이동 경로/방문 커버리지 점검
- 관리자: 정책 승인/변경 이력 관리

## 범위
- 포함: 입력 통합, 맵 마스터 생성, payload 생성, 템플릿 렌더, 로그 저장
- 제외: 신규 지오코딩 구축, 지도 프레임워크 교체

## 기능 요구사항
- FR-1: 2단계 생성(마스터 CSV -> 최종 HTML)
- FR-2: 좌표/키 품질 게이트 Fail-Fast
- FR-3: run_id 기반 구조화 로그 저장
- FR-4: 날짜 기반 파일명 + 충돌 회피 suffix
- FR-5: 동일 입력/동일 config 재현성 보장

## 비기능 요구사항
- NFR-1: UTF-8 고정
- NFR-2: 원본/기존 산출물 보존
- NFR-3: 실패 원인 로그만으로 재현 가능

## KPI
- 맵 좌표 결측률 <= 10%
- 병원키 미매핑률 <= 2%
- 맵 생성 성공률 >= 95%
- 평균 생성 시간 <= 10분

## 승인 기준
- 샘플/실데이터에서 CSV/HTML 생성 성공
- 품질 게이트 정상 작동
- 로그 필수 필드 누락 없음
